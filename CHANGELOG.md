# Changelog

All notable changes will be documented here. The project follows Semantic Versioning after the first public release.

## [Unreleased]

### Added

- Tolerant Turborepo run-summary discovery and parsing.
- Deterministic root-cause, cascade, and same-hash cache-miss diagnosis.
- Human, JSON, and GitHub-flavored Markdown reports.
- Optional local Git diff statistics.
- Cross-platform Cargo and npm release automation.
- Root-first topological cascade ordering and unchanged-input summaries.
- Curated hints for volatile CI variables, generated outputs, log inputs, lockfile-only changes, and missing output configuration.
- Real Turborepo 1.9 and current-v2 fixtures, exact renderer golden tests, parser mutation tests, and a multi-version canary.
- Automatic no-task baseline capture from task names declared in `turbo.json`.
- Branded architecture artwork and a static capture derived from the real terminal demonstration.

### Changed

- Updated every direct Rust dependency to its current stable registry release.
- Expanded the GitHub and npm README into a complete installation, usage, evidence, CI, compatibility, privacy, architecture, development, and release guide.
